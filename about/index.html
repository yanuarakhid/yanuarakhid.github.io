Sebenarnya sudah lama saya ingin membuat blog pribadi namun belum ada alesan yang jelas karena apa ingin membuat blog pribadi karena sebelumnya saya sudah memiliki blog baru juga yang berdomain zonabiner.com yang saya buat menggunakan ssg (static site generator) Jekyll yang berisi seputar Dunia IT khusunya seputar Komputer Jaringan, Sistem Operasi, Cloud Computing, dan Cyber Security dari hasil eksplorasi dan pembelajaran saya namun masih jarang menulis :) ,sampai akhirnya dibangku kuliah bertemu dengan mata kuliah Pemograman Web yang hasil akhir di matakuliah ini targetnya adalah bisa membuat Web Pribadi yang bisa dibuat dengan cara native atau dengan menggunakan css framework seperti Bootstrap. Blog Dibuat pakai apa ?

Pasti ada yang tanya gitu ya? atau mungkin tidak hehehhe, blog ini dibuat dengan menggunakan Nuxt.JS (Builds on top of Vue.JS) dan postingan ditulis dengan format Markdown untuk yang sudah tau didunia per-github an pasti sudah tidak asing. Untuk yang belum mengetahui apa itu markdown bisa dibaca-baca sendiri ya hehehe :) Mengapa memilih Nuxt.JS? Karena mendukung banyak fitur diantaranya SPA (Single Page Application), SSR (Static Site Rendering), dan tentunya juga sudah bisa support PWA (Progresive Web App) , Masih banyak lagi, dan tentunya kalau teman saya bilang anti reload-reload club hehehhe sebenarnya diluar banya platform lain yang sudah menyediakan SSG (Static Site Generator) yang ditulis dengan bahasa Javascript seperti (Gatsby.JS(Frontend React.JS),Gridsome(Frontend Vue.JS)). Walaupun ranah konsen pembelajaran saya nantinya tidak kesitu setidaknya saya tau perkembangan dan tidak ada salahnya untuk Belajar. Dengan membuat blog ini saya menjadi sedikit lebih tahu mengenai bahasa pemograman Javascript.

Spesifikasi Mesin

Berikut ini merupakan Mesin (Perangkat Komputer) yang perah saya gunakan dari sewaktu saya SMP dan saya gunakan hingga saat ini. Sedikit cerita sewaktu masih di SMK saya mulai menggunakan Linux dengan Sitem Operasi Ubuntu Dual Boot dengan Windows 10 (saya sangat merekomendasikan ubuntu bagi setiap orang yang ingin mulai belajar dalam menggunakan Linux) kemudian berpindah menggunakan Debian, lalu semenjak lulus dari SMK hingga saat ini saya Single Boot menggunakan Sistem Operasi Arch Linux. Mengapa menggunakan Linux ? saya menggunakan linux karena selain dia sistem operasi gratis tak berbayar juga karena aplikasi aplikasi penunjang kegiatan saya sehari-hari akan maksimal ketika menggunakan environtment linux (Seperti kegiatan, Konfigurasi Remote, Virtualisasi menggunakan (Virtualbox, Qemu, dan VMware), Docker, dan Coding/Development). Terlebih di OS yang saya gunakan saat ini paket aplikasi yang desediakan sangat banyak dan kita cukup install yang kita perlukan saja berkat Repo AUR (Arch User Repository).
